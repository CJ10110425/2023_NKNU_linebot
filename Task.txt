Task Title: Creation of Professor Database

Task Description:
The goal of this project is to construct a detailed professor database. This involves creating a structured database within our system for efficiently storing and retrieving professor information.

Objectives:
1. Develop a structured and accessible database for storing professor information.
2. Implement functionalities for inputting and retrieving professor data to enhance data management and accessibility.

Specific Requirements:
1. The database, named `professors_DB`, should include the following information for each professor:
   - Name
   - Gender
   - Subject Taught
2. The database should offer the following functionalities:
   - Function to input professor data.
   - Function to retrieve professor data.
   - Please feel free to discuss with me regarding the parameters and details of these functions.

Deadline: [11/12]

Git Instructions for Collaborators:
To clone the `professors_DB` branch to your local system, please use the following steps:

1. Open your terminal.

2. Clone the specific `professors_DB` branch with the command:
   `git clone -b professors_DB --single-branch https://github.com/kjdevelop/2023_Professor_DB.git`

3. Navigate to the cloned directory:
   `cd 2023_Professor_DB`

4. Make your changes in this branch. When you are ready to upload your changes to GitHub, execute the following commands:
   `git add .`
   `git commit -m "Your detailed commit message"`
   `git push origin professors_DB`

Ensure that your commit message precisely describes the changes you have made. If you face any issues, do not hesitate to reach out for assistance.
