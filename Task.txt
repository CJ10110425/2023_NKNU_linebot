Task Title: Creation of Student Database for Third-Year Electrical Engineering Students at National Kaohsiung Normal University

Task Description:
The objective is to develop a comprehensive student database for the third-year Electrical Engineering students at National Kaohsiung Normal University. This task entails creating a structured database within our system that efficiently stores and retrieves student information.

Objectives:
1. Create a structured and accessible database for storing student information.
2. Implement functions for inputting and retrieving student data, enhancing data management and accessibility.

Specific Requirements:
1. The database, named `students_DB`, should include the following information for each student:
   - Name
   - Gender
   - Student ID
2. The database should provide the following functionalities:
   - Function to input student data.
   - Function to retrieve student data.
   - Feel free to discuss with me regarding the parameters and details of these functions.

Deadline: [11/16]

Git Instructions for Collaborators:

1. Open your terminal.

   `git clone -b students_DB --single-branch https://github.com/kjdevelop/2023_NKNU_linebot.git`

   `cd 2023_NKNU_EE`

4. Make your changes in this branch. When you are ready to upload your changes to GitHub, use the following commands:
