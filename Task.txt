Task Title: Rich Menu Design and Dynamic Switching Functionality

Task Description:
We require a dynamic Rich Menu interface to enhance the user interaction experience for our academic platform. This task involves creating a rich menu with specific features for professors and implementing a feature that allows the dynamic switching of menus based on user actions.

Objectives:
1. Design a Rich Menu that is in line with our linebot platform's branding and user experience.
2. Develop a Rich Menu switching functionality that allows professors to switch between different menu options based on the context (initializing, distributing homework, announcing exams, handling student questions).

Specific Requirements:
1. The Rich Menu for professors should include the following three functionalities:
   - `distribute_homework`
   - `announce_exam`
   - `handle_student_question`
2. An initial Rich Menu view with a single option:
   - `initialize`

Deadline: [11/12]
